# Savvy — Rule-Based AI Chatbot
**DecodeLabs Industrial Training | Batch 2026 | Project 1**

A rule-based AI chatbot that asks for your name and personalizes 
every response. Includes two mini-games: Number Guessing and AI Trivia.

## Files
- `chatbot.py` — terminal version
- `savvy.html` — browser UI with dark/light mode

## How to run
- Terminal: `python3 chatbot.py`
- Browser: open `savvy.html` directly
